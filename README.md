# SmartCards

**SmartCards** — это современное веб-приложение для интервального повторения (Spaced Repetition System) материала. Приложение помогает эффективно запоминать информацию, автоматически планируя время следующего повторения для каждой карточки на основе алгоритма SM-2.

## Демонстрация

Попробовать приложение можно здесь:
[https://DrobyninArtiom.github.io/SmartCards/](https://DrobyninArtiom.github.io/SmartCards/)

## Особенности

- **Алгоритм SM-2**: Проверенная временем математическая модель для оптимального запоминания.
- **Интерфейс**: Современный и адаптивный дизайн в темной теме.
- **Управление наборами**: Создание, редактирование и удаление наборов карточек.
- **Статистика**: Наглядный дашборд с прогрессом обучения (Новые, Изучаю, На повторение).
- **Импорт/Экспорт**: Возможность сохранять и загружать наборы в формате JSON.

## Технический стек

- **React 18** + **TypeScript**
- **Vite** (билд и dev-сервер)
- **Tailwind CSS** (стилизация)
- **Lucide React** (иконки)
- **ESLint** (контроль качества кода)

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/DrobyninArtiom/SmartCards.git
    cd SmartCards
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Запустите локальный сервер:**
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу `http://localhost:5173`.

## Сборка

Для создания оптимизированной production-версии:

```bash
npm run build
```

Файлы сборки появятся в папке `dist`.
